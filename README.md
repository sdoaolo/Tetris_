# Tetris_
2018년 C언어 기말고사로 제작한 테트리스입니다.

### Function 💜
1. 테트리스 블럭 6개
2. ID 입력을 통한 점수기록
3. 레벨업 할때마다 내려오는 속도가 빨라짐
4. Next BLOCK 미리보기
5. Block Shadow
6. 지금까지 기록 된 점수 중 Best Score를 보여줌
7. 세로로 한줄을 날려버리는 폭탄
8. 한번에 없애는 줄 개수(1~4)에 따른 점수 차등 지급
